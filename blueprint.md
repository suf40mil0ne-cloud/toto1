# 로또리듬(Lotto Rhythm) 프로젝트 블루프린트

## 프로젝트 개요
로또리듬은 사용자의 바이오리듬을 분석하여 개인화된 로또 번호 조합을 생성하고, 최신 회차 정보 및 판매점 당첨 통계를 제공하는 웹 서비스입니다. 기능적인 데이터 제공뿐만 아니라 상세한 가이드를 통해 사용자가 로또 데이터를 올바르게 해석하도록 돕는 것을 목표로 합니다.

## 주요 기능 및 특징
- **바이오리듬 분석:** 생년월일을 기반으로 신체, 감성, 지성 리듬을 시각화하고 점수화합니다.
- **번호 생성기:** 분석된 바이오리듬 가중치를 반영하여 참고용 로또 번호 세트를 생성합니다.
- **최신 회차 조회:** 직전 회차의 당첨 번호, 당첨금, 실수령액 정보를 실시간으로 확인합니다.
- **판매점 조회:** 특정 회차의 1등 배출 판매점 정보를 조회합니다.
- **데이터 가이드:** 통계 해석법, 사이트 이용법 등 상세한 교육형 콘텐츠를 제공합니다.
- **SEO & AdSense 최적화:** 구글 에드센스 심사 및 검색 엔진 최적화를 위한 구조가 마련되어 있습니다.

## 현재 구현 상태
- HTML5, CSS3(Baseline), JavaScript(ES Module) 기반의 정적 웹 사이트.
- Firebase Cloud Functions를 통한 데이터 API 연동 (`/api/draw`, `/api/stores`, `/api/generate`).
- 현대적인 카드 레이아웃과 반응형 디자인 적용.

## 디자인 업그레이드 계획 (Stitch 스타일 적용)
기존의 기능과 AdSense 설정을 유지하면서, 디자인을 더욱 세련되고 인터랙티브하게 업그레이드합니다.

### 1. 시각적 스타일 개선
- **색상 체계:** `oklch`를 사용하여 더욱 생생하고 현대적인 컬러 팔레트로 전환.
- **유리 효과(Glassmorphism):** 헤더, 카드, 버튼 등에 `backdrop-filter`와 미세한 경계선을 적용하여 깊이감 부여.
- **타이포그래피:** 가독성이 뛰어난 IBM Plex Sans KR과 Manrope 폰트의 계층 구조 강화.
- **질감(Texture):** 배경에 미세한 노이즈 텍스처를 추가하여 프리미엄 느낌 강조.

### 2. 컴포넌트 고도화
- **Interactive Cards:** 호버 시 미세한 움직임과 글로우(Glow) 효과 추가.
- **Advanced Shadows:** 다층 레이어 그림자를 사용하여 요소가 떠 있는 듯한 느낌 구현.
- **Container Queries:** 부모 요소의 크기에 따라 유연하게 반응하는 컴포넌트 설계.

### 3. 애니메이션 및 효과
- **Smooth Transitions:** 상태 변화 시 부드러운 전환 효과 적용.
- **Loading States:** 데이터 로딩 시 세련된 스켈레톤 또는 스피너 효과 보강.

## 실행 단계
1. **[Research]** 기존 `style.css`와 `index.html` 구조 분석 (완료).
2. **[Strategy]** 디자인 업그레이드 계획 수립 및 블루프린트 작성 (진행 중).
3. **[Execution]** `style.css` 전면 리뉴얼 및 `index.html` 보정.
4. **[Validation]** 디자인 완성도 및 기존 기능 작동 여부 확인.
