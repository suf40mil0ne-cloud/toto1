# 로또리듬(Lotto Rhythm) 프로젝트 블루프린트

## 프로젝트 개요
로또리듬은 사용자의 바이오리듬을 분석하여 개인화된 로또 번호 조합을 생성하고, 최신 회차 정보 및 판매점 당첨 통계를 제공하는 웹 서비스입니다. 기능적인 데이터 제공뿐만 아니라 상세한 가이드를 통해 사용자가 로또 데이터를 올바르게 해석하도록 돕는 것을 목표로 합니다.

## 주요 기능 및 특징
- **바이오리듬 분석:** 생년월일을 기반으로 신체, 감성, 지성 리듬을 시각화하고 점수화합니다.
- **번호 생성기:** 분석된 바이오리듬 가중치를 반영하여 참고용 로또 번호 세트를 생성합니다.
- **최신 회차 조회:** 직전 회차의 당첨 번호, 당첨금, 실수령액 정보를 실시간으로 확인합니다.
- **판매점 조회:** 특정 회차의 1등 배출 판매점 정보를 조회합니다.
- **데이터 가이드:** 통계 해석법, 사이트 이용법 등 상세한 교육형 콘텐츠를 제공합니다.
- **SEO & AdSense 최적화:** 구글 에드센스 심사 및 검색 엔진 최적화를 위한 구조가 마련되어 있습니다.

## 현재 구현 상태
- HTML5, CSS3(Baseline), JavaScript(ES Module) 기반의 정적 웹 사이트.
- Firebase Cloud Functions를 통한 데이터 API 연동 (`/api/draw`, `/api/stores`, `/api/generate`).
- 현대적인 카드 레이아웃과 반응형 디자인 적용.

## 디자인 업그레이드 계획 (Stitch Premium Dark 적용)
기존의 에메랄드 테마에서 한 단계 더 진화하여, 세련된 **다크 모드(Deep Obsidian)**와 **일렉트릭 인디고(Electric Indigo)** 포인트 컬러를 적용한 프리미엄 AI 스타일로 전환했습니다.

### 1. 시각적 스타일 개선 (Dark Evolution)
- **색상 체계:** `oklch`를 사용하여 깊이 있는 검은색과 생생한 인디고 블루의 대비를 강조. 배경은 Deep Obsidian, 주요 액션은 Electric Indigo 적용.
- **유리 효과(Glassmorphism):** 다크 테마에 맞춰 헤더와 카드 요소의 투명도와 블러 효과를 강화하여 더 깊이 있는 층 레이어링 구현.
- **타이포그래피:** 가독성이 뛰어난 IBM Plex Sans KR 유지, 제목 요소에 미세한 그라데이션 텍스트 효과 적용.
- **글로우 효과(The Glow):** 주요 버튼과 인터랙티브 카드에 부드러운 인디고 글로우를 추가하여 현대적인 AI 서비스 느낌 강조.

### 2. 레이아웃 재구조화
- **Vertical Flow:** 기존의 좌우 2열 구조를 시선 흐름이 자연스러운 1열 세로 스택으로 변경.
- **Hero Stats:** 가장 중요한 '직전 회차 실수령액'을 히어로 섹션 바로 아래에 배치하여 정보 우선순위 최적화.
- **Responsive Steps:** 시작 단계(Quick Steps)를 카드 그리드 형태로 하단에 배치하여 모바일에서도 완벽하게 대응.

### 3. 컴포넌트 세부 조정
- **로고 리디자인:** 미니멀한 기하학 심볼과 정제된 워드마크 조합으로 변경.
- **카드 인터랙션:** 호버 시 `z-index`를 조정하여 요소 겹침 문제를 해결하고, 부드러운 부유 효과(Lifted effect) 적용.
- **모바일 최적화:** 폰트 크기, 간격, 컨트롤 패널 너비 등을 모바일 환경에 맞춰 전면 재조정.

## 실행 단계
1. **[Research]** 기존 `style.css`와 `index.html` 구조 분석 (완료).
2. **[Strategy]** 디자인 업그레이드 계획 수립 및 블루프린트 작성 (진행 중).
3. **[Execution]** `style.css` 전면 리뉴얼 및 `index.html` 보정.
4. **[Validation]** 디자인 완성도 및 기존 기능 작동 여부 확인.
